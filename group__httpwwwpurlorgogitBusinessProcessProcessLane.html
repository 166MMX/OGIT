ProcessLane
Entities

ogit/BusinessProcess/ProcessLane

Entity to represent a ProcessLane of a business process. There must be one or more Lanes within a Pool. If there is only one Lane, then that Lane shares the name of the Pool and only the Pool name is displayed. If there is more than one Lane, then each Lane has to have its own name and all names are displayed.
